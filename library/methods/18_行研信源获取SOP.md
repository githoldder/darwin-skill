## 目标
在30分钟内定位并获取目标行业最新、最权威数据，支持决策/报告/竞品分析，避免二手解读偏差。

## 前置条件
- 明确研究主题（行业+时间+维度：市场规模/用户行为/竞争格局/政策等）
- 准备好企业/学校邮箱（部分报告需验证）

## 标准操作流程（4步法）

### Step 1: 官方一手数据（5分钟，必做）
1. 国家统计局 / 工信部 / CNNIC 官网搜索最新年报/统计公报
2. 对应行业协会官网下载白皮书/年度报告
3. 记录数据来源+发布日期（作为报告底稿）

### Step 2: 专业机构核心报告（10分钟）
1. 艾瑞报告.iresearch.cn → 搜索关键词 → 购买/会员下载
2. 易观analysys.cn → 订阅产品/免费试用区
3. 麦肯锡mckinsey.com.cn/insights → 直接免费PDF
4. 补充：QuestMobile / 头豹 / CBNData（同上逻辑）

### Step 3: 辅助验证数据（10分钟）
1. WIND/东方财富 → 券商研报 + 上市公司年报
2. 百度指数 / 阿里研究院 / 巨量算数 → 实时指数验证
3. CNKI → 学术论文交叉印证

### Step 4: 存档与复用机制
- 文件命名规范：`[年份]_[机构]_[行业]_[主题].pdf`
- obsidian数据库记录：来源、链接、关键数据摘录、更新日期
- 每月订阅机构公众号/邮件，自动接收最新报告

## 质量校验Checklist
- [ ] 数据发布时间≤6个月
- [ ] 来源为官网/官方机构（非知乎/小红书总结）
- [ ] 多源交叉验证（至少2家机构数据一致）
- [ ] 付费报告优先于免费摘要

## 异常处理
- 报告付费无法获取 → 优先找学校图书馆/企业内部分享 → 退而求其次用官方数据+公开摘要
- 数据冲突 → 以政府/协会数据为准

## 输出物
每次行研结束输出《信源清单表》（Markdown），作为所有报告/方案的附录。

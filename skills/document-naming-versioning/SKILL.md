# Document Naming Versioning

## Workflow

1. Choose a filename pattern by document family.
2. Encode type, scope, version, and date consistently.
3. Apply semantic versioning rules to document releases.
4. Maintain a changelog for meaningful revisions.
5. Keep archive folders and references aligned with the version model.

## Quality Checks

- File names are predictable and sortable.
- Version jumps have a reason.
- Formal documents have traceable revisions.

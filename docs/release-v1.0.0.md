# Darwin Skill v1.0.0

This is the first package-ready release of `darwin-skill`.

## Highlights

- Defines Darwin Skill as a local AI agent skill-evolution plugin.
- Adds Full and Pure package profiles.
- Adds `v1.0.0` version metadata.
- Adds project logo.
- Documents author-provided skills separately from Darwin core skills.
- Keeps a Pure package for users who only want the Darwin mechanism.

## Packages

- `darwin-skill-v1.0.0.zip`: Full package with Darwin core, author-provided skills, scripts, registry, and library references.
- `darwin-skill-pure-v1.0.0.zip`: Pure package with only Darwin core skills and supporting scripts/templates.

- `darwin-skill-manager`
- `darwin-skill-distiller`
- `darwin-skill-auditor`

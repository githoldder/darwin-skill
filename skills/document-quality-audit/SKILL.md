# Document Quality Audit

## Audit Dimensions

- structure completeness
- factual accuracy
- logical coherence
- standards compliance
- evidence strength
- executability
- maintainability
- readability
- traceability

## Workflow

1. Identify document type and intended use.
2. Check any disqualifying issues first.
3. Score or judge the document across the relevant dimensions.
4. Separate blocking issues from polish issues.
5. Produce a concise review with fixes ranked by severity.

## Output

Return:

- current readiness
- blocking issues
- important improvements
- release or revision recommendation
